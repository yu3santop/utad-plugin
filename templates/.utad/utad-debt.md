## How to Use This File

1. **New entries** are added automatically by `.claude/hooks/post-edit-check.sh`
2. **Review** entries during `/utad-sync` — prioritise and assign
3. **Resolve** by writing a Test Instance with `/utad-test`, then checking `[ ]` → `[x]`
4. **Archive** resolved entries at the bottom of this file monthly
